# Image assets

Local backup copies of the photography used across the demo collection.

**Live status:** the HTML currently loads these from the Unsplash CDN
(`images.unsplash.com/photo-{ID}?...`) for speed and auto-format (WebP/AVIF).
These local files are the offline / fallback copies. To switch a site to local
images, replace the CDN `src` with `assets/images/<file>` (adjust the relative
path per folder depth — e.g. `../assets/images/...` from a site subfolder).

**License:** Unsplash License — free for commercial use, no attribution required.
Downloaded at `fit=crop&w=1200&q=80`.
